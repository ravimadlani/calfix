Option 1: Compact Horizontal (All Metrics Inline)

Pros: Most compact (single row), shows all 5 key factors at once

Cons: Can feel cramped, harder to add more factors in future, needs horizontal scroll on mobile

Best for: Power users who want quick glance at all metrics

Option 2: Two-Row Grid (8 Factors Total)

Pros: Shows more factors (8 total), organized layout, more breathing room

Cons: Takes more vertical space, includes inactive (0 value) factors

Best for: Complete transparency - users see all factors being tracked

Option 3: Expandable (Smart Display)

Pros: Compact by default (only shows active factors), can expand to see all, adapts to data

Cons: Requires interaction to see all factors, more complex to implement

Best for: Balanced approach - stays clean but allows full visibility when needed

Recommendation:

Option 3 (Expandable) - Shows only factors that matter today (non-zero) by default, keeping the UI clean and focused. Users can expand to see all tracked factors if they want full transparency. This adapts to the data naturally.
